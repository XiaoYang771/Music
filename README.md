该项目是一个基于 Vue 3 + TypeScript 开发的音乐 Web 应用。
  一.核心功能:
    1.用户系统:包含登录、注册功能，通过 localStorage 存储用户信息，登录页面前会验证登录状态（未登录时跳转至登录页）。
    2.音乐播放:支持歌曲播放、暂停、切换上/下一首，维护播放列表，可添加喜欢的歌曲到“我的喜欢”列表。
    3.歌单管理:
      - 展示歌单广场，支持查看歌单详情
      - 用户可创建自定义歌单，包含设置歌单名称、上传歌单图片
      - 支持向歌单中添加歌曲
    4.搜索功能:可搜索歌曲并进行相关操作（如添加到播放列表、歌单等）。
    5.MV播放:支持 MV 播放及相关 MV 推荐，可切换不同 MV 观看。
    6.歌词展示:播放歌曲时同步显示歌词，并有高亮当前歌词的效果。
    7.个人中心:包含用户信息展示、我的笔记（可发布和查看）、创建的歌单等内容。
  二.技术栈与工具:
    1.前端框架：Vue 3（使用 `<script setup>` 语法）。
    2.状态管理：Pinia（用于管理音乐列表、播放状态、歌单等数据）。
    3.路由：Vue Router（配置了多种页面路由，包含路由守卫验证登录）。
    4.样式：SCSS（使用 scoped 样式隔离组件样式）。
    5.构建工具：Vite。
    6.类型检查：TypeScript（定义了歌曲、歌单等接口类型）。
  三.项目结构:
    1.组件（components）:包含音乐播放控制栏、歌单列表、用户中心、搜索组件等可复用组件。
    2.视图（views）:包含登录页、首页、歌单中心、添加歌曲页等页面级组件。
    3.路由（router）:配置应用路由及路由守卫。
    4.状态存储（stores）:使用 Pinia 定义音乐和歌单相关的状态管理。
    5.静态资源:包含图片、视频等资源文件。
